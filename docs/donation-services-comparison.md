# 寄付サービス比較 - 個人開発者向け

このドキュメントは、四則（Shisoku）ゲームに寄付リンクを設置するための、主要な寄付サービスの比較です。

## TL;DR（結論）

### 🥇 最推奨: **Ko-fi**

**理由**: プラットフォーム手数料0%、セットアップ5分、海外・日本両対応

| 項目 | 詳細 |
|---|---|
| **手数料** | 0%（チップ） + 決済手数料のみ（実効9%） |
| **月額** | 無料 |
| **セットアップ** | 5分 |
| **実装** | HTMLタグ貼るだけ |

### 💰 収益シミュレーション

- **月1万PV（支援率0.1%）**: 約¥2,000/月
- **月10万PV（支援率0.2%）**: 約¥45,000/月

### 🎯 次のステップ

1. Ko-fiでアカウント作成（https://ko-fi.com）
2. ウィジェットコード取得
3. EndScreenに埋め込み
4. GitHub Pagesにデプロイ

## 詳細比較

### 1. Ko-fi ⭐⭐⭐⭐⭐（最推奨）

#### 特徴
- **プラットフォーム手数料0%**（チップ・寄付の場合）
- シンプルで使いやすいUI
- 簡単に埋め込み可能なボタン
- 日本からの支援も受け取り可能

#### 手数料構造
```
チップ（一回限りの寄付）: 0%
メンバーシップ: 5%（Goldプランなら0%）
商品販売: 5%（Goldプランなら0%）
決済手数料: 2.9% + $0.30（Stripe/PayPal）
```

#### 実際の受取額（$5のチップの場合）
```
支援額: $5.00
Ko-fi手数料: $0（0%）
決済手数料: $0.30（2.9% + $0.30 = $0.45）
受取額: $4.55（91%）
```

#### メリット
- ✅ 最も手数料が安い（チップは0%）
- ✅ セットアップが簡単（5分で完了）
- ✅ 埋め込みボタンのカスタマイズ可能
- ✅ 月額固定費なし
- ✅ 国際的に利用されている

#### デメリット
- ⚠️ 英語中心（日本語UIはない）
- ⚠️ 最低支払額の設定が$3から

#### 実装方法
1. Ko-fiでアカウント作成
2. ボタンのコード取得
3. HTMLに埋め込み

```html
<script type='text/javascript' src='https://storage.ko-fi.com/cdn/widget/Widget_2.js'></script>
<script type='text/javascript'>
  kofiwidget2.init('Support Me on Ko-fi', '#29abe0', 'YOUR_USERNAME');
  kofiwidget2.draw();
</script>
```

## 推奨構成

### 個人ゲーム開発者向けベストプラクティス

あなたのゲーム（四則パズル）の場合、以下の組み合わせを推奨します：

#### パターンA: シンプル構成（最推奨）

**Ko-fi 1つのみ**
```
メリット:
- 手数料0%（チップ）で最安
- 海外・日本両方からの支援OK
- セットアップ5分
- メンテナンス不要

コスト: $0
手数料: 0%（チップ） + 決済手数料のみ
```

#### パターンB: 開発者向け構成

**Ko-fi + GitHub Sponsors**
```
Ko-fi: 一般ユーザー・カジュアル支援向け
GitHub Sponsors: 開発者コミュニティ向け

メリット:
- 両方とも手数料0%
- ターゲット層を分けられる
- プロフェッショナルな印象

デメリット:
- GitHub Sponsorsの審査に1〜2週間
```

#### パターンC: 日本特化構成

**Ko-fi + Amazon欲しいものリスト**
```
Ko-fi: 海外・現金支援向け
Amazon: 日本・物品支援向け

メリット:
- 日本ユーザーの選択肢が増える
- Amazon欲しいものリストは手数料0%

デメリット:
- 管理するサービスが増える
```

## 実装例: Ko-fi ボタンをゲームに追加

### HTMLに埋め込む場合

```html

<script type='text/javascript' src='https://storage.ko-fi.com/cdn/widget/Widget_2.js'></script>
<script type='text/javascript'>
  kofiwidget2.init('☕ Support this game', '#29abe0', 'YOUR_USERNAME');
  kofiwidget2.draw();
</script>

<!-- または、シンプルなリンクボタン -->
<a href='https://ko-fi.com/YOUR_USERNAME' target='_blank'>
  <img height='36' style='border:0px;height:36px;'
       src='https://storage.ko-fi.com/cdn/kofi2.png?v=3'
       border='0' alt='Buy Me a Coffee at ko-fi.com' />
</a>
```

### React コンポーネントとして追加

```tsx
// components/DonationButton.tsx
export const DonationButton: React.FC = () => {
  useEffect(() => {
    const script = document.createElement('script');
    script.src = 'https://storage.ko-fi.com/cdn/widget/Widget_2.js';
    script.async = true;
    document.body.appendChild(script);

    script.onload = () => {
      // @ts-ignore
      if (window.kofiwidget2) {
        // @ts-ignore
        window.kofiwidget2.init('Support this game', '#29abe0', 'YOUR_USERNAME');
        // @ts-ignore
        window.kofiwidget2.draw();
      }
    };

    return () => {
      document.body.removeChild(script);
    };
  }, []);

  return <div id="kofi-widget-container"></div>;
};
```
