# Shisoku Problem Generator

四則演算パズルゲーム「四則」の問題生成ツール

## 概要

このツールは、ゲームで使用する問題データベース（`../src/data/problems.json`）を生成します。

## セットアップ

```bash
npm install
```

## 使い方

### テスト実行

```bash
npm test
```

### 問題生成（2,000問）

```bash
npm run generate:2000
```

生成された問題は `../src/data/problems.json` に直接出力されます。

## 問題の分類

- **no-parens**: 括弧なしで解ける問題（例: `1 + 2 + 3 + 4 = 10`）
- **one-paren**: 括弧1つで解ける問題（例: `(1 + 2) * 3 + 4 = 13`）
- **multi-paren**: 複数の括弧が必要な問題（例: `((1 + 2) * 3) + 4 = 13`）

## 除外される問題（トリビアル問題）

- 目標が0 かつ 数字に0が含まれる
- 全ての数字が同じ（例: `[2, 2, 2, 2]`）

## 生成目標

- no-parens: 800問
- one-paren: 800問
- multi-paren: 400問
- **合計: 2,000問**
